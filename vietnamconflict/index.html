U.S. Army Casualties

The Vietnam War was a long, costly and divisive conflict between the communist government of North Vietnam and South Vietnam and its principal ally, the United States. It lasted from 1955 to the fall of Saigon in 1975, with more than 58,000 Americans killed in the war. This data visualization demonstrates the burden of casualties that the US Army bore in Vietnam.

About

The data used for this analysis comes from:

  • The Vietnam Conflict Extract Data File of the Defense Casualty Analysis System (DCAS) Extract Files contains records of 58,220 U.S. military fatal casualties of the Vietnam War. It is accessible for direct download via the U.S National Archives Catalog.
  • This dataset was taken from Kaggle where the raw data was parsed, cleaned and labelled with reference to the accompanying Supplemental Code Lists. I have processed the data further by recategorizing province names.

The realization of this visual data project was performed using several technologies:

  • Data preparation and analysis was done in Python using Jupyter notebook.
  • The visuals were created using D3.js. Canvas was used to render the dotplot.
  • The UI framework used is Semantic UI.

Special thanks goes to:
