How to create animated SVG Arc Paths

Posted on 4 August, 2019

Render paths

Code

Let's start off with straight lines. Create 12 'spokes' around a point. SVG lines are straightforward to create, you simply need to populate the x1, y1, x2, y2 attributes of a line element

  • The x1 attribute defines the start of the line on the x-axis
  • The y1 attribute defines the start of the line on the y-axis
  • The x2 attribute defines the end of the line on the x-axis
  • The y2 attribute defines the end of the line on the y-axis

Creating a smooth arc between two points is much more challenging.
