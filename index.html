Formula 1

Created a dockerized full-stack web application, with a Python Flask server and a React front-end to showcase my D3 visualizations of Formula 1 race results.

  • Backend stack: Python 3.5+, Flask, SQLAlchemy, Celery
  • Production: PostgreSQL, Redis, Docker, Amazon Web Services
